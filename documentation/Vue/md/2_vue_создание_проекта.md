# Как создать проект на Vue

Здесь рассматривается информация о том, как создать проект на Vue.

1. Откройте терминал и выполните следующую команду:

   ```bash
   npm create vite@7.0.0
   ```

   _При вызове этой команды, вам, возможно, предложат установить сборщик Vite. Соглашаемся, нажимая **Y**._

   Далее необходимо настроить проект при его создании.

2. В качестве `Project name` введите название создаваемого проекта, например `vue-project`.

3. В качестве `Select a framework` выберите `vue`.

4. В качестве `Select a variant` выберите `TypeScript`, чтобы создать проект с поддержкой TypeScript.  
   _После этапа 4 могут быть дополнительные настройки, их лучше не использовать и оста��ить все по умолчанию._

5. После настройки проекта, перейдите в его директорию и доустановите пакеты:

   ```bash
   cd <название созданного проекта>
   npm install
   ```
