# Как установить Vue.

Здесь рассматривается информация о том, как установить Node JS для разработки на Vue.

1. Для начала нужно установить Node.js, который является средой выполнения JavaScript и позволяет использовать npm (Node Package Manager) для установки Vue и других пакетов.  
   Рекомендуется установить последнюю LTS (Long Term Support) версию для стабильной работы.

   **Для Windows:** Скачать Node.js можно с официального сайта: [https://nodejs.org/en/download](https://nodejs.org/en/download).

   **Для Linux:** Скачать Node.js можно, используя ваш пакетный менеджер, например:

   ```bash
   sudo apt install nodejs
   ```

   ```bash
   sudo dnf install nodejs
   ```

2. Убедитесь, что Node.js и npm установлены правильно, выполнив следующие команды в терминале:

   ```bash
   node --version
   npm --version
   ```

3. Установите Vue CLI для запуска команд Vue. Это можно сделать с помощью пакетного менеджера npm:
   ```bash
   npm install -g @vue/cli
   ```
